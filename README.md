## Dependencies
- Node.js 
- Typescript
- Yarn

## Running Project
The following commands are in the package.json for reference:
#### Install dependencies into /node_modules
```
yarn install
```
#### Compile Typescript into Javascript in the /dist folder
```
yarn build
```
#### Start the code by running the index.ts file
```
yarn start
```
